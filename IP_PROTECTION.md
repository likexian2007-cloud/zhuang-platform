# 装配智建·和美乡村 知识产权与安全保护清单

> 本清单用于项目答辩、交付和正式登记准备，不构成法律意见。商标、版权、软件著作权登记需要由权利人或受托代理机构提交真实身份材料并签章。

## 1. 平台内置保护

- 授权访问码：公开页面进入平台前需要输入项目访问码，访问码只以 SHA-256 哈希形式保存在前端。
- 版权水印：页面右下角和背景叠加“内部演示资料 · 未经授权禁止复制传播”标识。
- 防复制提示：拦截右键、拖拽、复制、保存页面、查看源码快捷键等常见复制动作，并弹出版权提示。
- 版权元信息：`index.html` 已写入 author、copyright、robots noarchive/noindex 信息。
- 后端安全头：Node 后端统一输出 `X-Content-Type-Options`、`X-Frame-Options`、`Referrer-Policy`、`Permissions-Policy`。
- 接口防刷：后端按 IP 做分钟级限流，默认每分钟 180 次请求。
- 可选 API Token：正式部署时设置 `PLATFORM_API_TOKEN` 后，AI、图纸识别、记录新增/修改/删除接口必须携带 `X-Platform-Token`。
- CORS 白名单：正式部署时通过 `ALLOWED_ORIGINS` 限制允许访问后端的域名。

## 2. 正式部署建议

GitHub Pages 是公开静态托管，任何前端代码最终都可能被查看。若项目资料需要真正保密，建议：

- 将仓库改为私有仓库，公开展示只放演示版。
- 使用 Vercel/Netlify/Cloudflare Pages + Serverless Functions 或独立云服务器承载后端。
- 后端设置：
  - `PLATFORM_API_TOKEN=<强随机口令>`
  - `ALLOWED_ORIGINS=https://你的正式域名`
  - `RATE_LIMIT=60`
- DeepSeek Key 只放在后端环境变量或 `backend/ai-config.json`，不得提交到仓库。
- 重要资料、PPT、源代码提交前打包存证，保留 Git 提交记录、创作过程截图和项目文件时间戳。

## 3. 软件著作权登记准备

适合登记名称建议：

- `装配智建·和美乡村装配式智慧建造管理平台`
- 软件简称：`装配智建平台`
- 版本号：`V1.0`

材料准备：

- 软件著作权登记申请表。
- 源程序鉴别材料：通常提交前、后各连续 30 页；不足 60 页提交全部源程序。
- 文档鉴别材料：用户手册、设计说明书或操作说明，通常前、后各连续 30 页；不足 60 页提交全部。
- 身份证明：个人身份证或企业营业执照等。
- 如有校企合作、委托开发、团队共创，应准备权属说明、合作协议或授权说明。

官方依据与入口：

- 国家版权局公布的作品登记机关和办理机构名录说明，作品登记机关包括国家版权局和各省级版权局，办理机构可受理作品登记申请；计算机软件著作权登记按《计算机软件著作权登记办法》执行。
- 中国版权保护中心：<https://www.ccopyright.com.cn/>

## 4. 商标注册准备

建议商标名称：

- `装配智建`
- `装配智建·和美乡村`
- `Prefab Rural Housing Digital Command Center` 可作为英文标识补充评估

建议优先检索类别：

- 第 9 类：计算机软件、可下载应用、数据处理设备相关。
- 第 42 类：软件即服务、平台即服务、工程技术研究、建筑数字化技术服务相关。
- 第 37 类：建筑施工、安装、修缮服务相关。
- 第 35 类：项目管理、商业管理辅助、数据管理展示相关。

正式提交前务必先做近似商标检索，避免与已有商标冲突。

官方入口：

- 国家知识产权局答复中说明，自行办理可通过商标网上服务系统在线提交商标注册申请，系统网址：<https://sbj.cnipa.gov.cn/sbj/wssq/>
- 国家知识产权局商标局咨询电话：`010-62356655` 按 2 键。
